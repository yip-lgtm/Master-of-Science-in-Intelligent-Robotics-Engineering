# MAEG3080 Fundamentals of Machine Intelligence

**Course Type**: Major Elective
**Units**: 3
**Stream**: Robotics and Automation / Mechatronics

## Official Description
Introduction to machine intelligence techniques for engineering applications.
Covers fundamental concepts in pattern recognition, neural networks, and
intelligent system design relevant to mechanical and automation engineering.

## Key Topics
- Pattern recognition fundamentals
- Artificial neural networks (ANN)
- Supervised and unsupervised learning
- Decision trees and ensemble methods
- Clustering algorithms
- Feature extraction and selection
- Application to mechanical systems

## Learning Outcomes
- Understand machine learning paradigms
- Apply ML to mechanical engineering problems
- Design intelligent control systems
- Evaluate algorithm performance
