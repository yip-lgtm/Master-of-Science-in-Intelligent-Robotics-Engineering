# MAEG4050 Modern Control Systems Analysis and Design

**Course Type**: Major Elective
**Units**: 3
**Stream**: Robotics and Automation / Mechatronics

## Official Description
Advanced control theory. State-space methods, optimal control, robust control,
and digital control system design. Builds on MAEG3050 (Intro to Control).

## Key Topics
- State-space representation
- Controllability and observability
- Pole placement design
- Linear Quadratic Regulator (LQR)
- Kalman filter (state estimation)
- Observer design (Luenberger)
- Robust control (H∞)
- Digital control and Z-transform
- Sampled-data systems

## Learning Outcomes
- Design state-feedback controllers
- Implement state estimators (observers)
- Apply LQR optimal control
- Analyze robustness and stability margins
- Convert between continuous and discrete systems
