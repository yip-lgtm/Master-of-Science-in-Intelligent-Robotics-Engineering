# ENGG2020 Digital Logic and Systems

**Course Type**: Major Required
**Units**: 3
**Stream**: All (Foundation for Digital Systems)

## Official Description
Digital logic design and computer organization. Combinational and sequential
logic, FSM (Finite State Machines), memory, processor architecture, and
embedded systems basics.

## Key Topics
- Boolean algebra and logic gates
- Combinational logic design
- Sequential logic and flip-flops
- Finite State Machines (FSM) - Moore/Mealy
- Memory (RAM, ROM, registers)
- Processor architecture basics
- Assembly language intro
- Embedded systems overview
- Microcontroller vs microprocessor

## Learning Outcomes
- Design combinational/sequential logic circuits
- Implement FSMs in hardware/software
- Understand CPU architecture
- Program simple embedded systems
