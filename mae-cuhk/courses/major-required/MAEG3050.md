# MAEG3050 - Introduction to Control Systems

## 📋 Course Info
- **Code:** MAEG3050
- **Title:** Introduction to Control Systems
- **Units:** 3
- **Term:** 5 (Year 2-3)
- **Category:** Major Required

## 🎯 Course Description
Analysis and design of feedback control systems. Modeling in time and
frequency domains, stability analysis (Routh-Hurwitz, Nyquist, Bode),
PID control, root locus, and compensator design.

## 📚 Topics
- Open-loop vs closed-loop systems
- Transfer functions and block diagrams
- Time response (1st order, 2nd order, steady-state error)
- Stability (Routh-Hurwitz, Nyquist, Bode)
- Root locus
- PID controller design
- Lead/Lag compensators
- State-space introduction

## 🔗 Prerequisites
- MAEG2020 Engineering Mechanics
- ENGG2740 Differential Equations
- ELEG2202 Electric Circuits

## 💡 Industry Applications
- Servo motor control (robotics!)
- Process control (chemical, manufacturing)
- Autopilot systems
- HVAC control
