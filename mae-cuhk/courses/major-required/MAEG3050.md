# MAEG3050 Introduction to Control Systems

**Course Type**: Major Required  
**Units**: 3

## Official Description
This course provides students with fundamental knowledge of control systems. It covers the following topics: mathematical modelling and linear approximation of engineering systems, Laplace transform, transfer function and block diagram representation, characteristics of feedback systems, performance specifications, Routh-Hurwitz stability criterion, root locus design, frequency response design and Nyquist criterion. The utilization of computer-aided analysis and design software is also included.

## Key Topics
- Mathematical modelling of engineering systems
- Laplace transform and transfer functions
- Block diagram reduction
- Feedback system characteristics and performance specifications
- Stability analysis (Routh-Hurwitz)
- Root locus design
- Frequency response and Nyquist criterion
- Computer-aided design (MATLAB / Python control libraries)

## Learning Outcomes
- Model physical systems mathematically
- Analyze stability and performance of feedback systems
- Design controllers using root locus and frequency response methods
- Use software tools for control system analysis and design

## Self-Study Notes & Resources
