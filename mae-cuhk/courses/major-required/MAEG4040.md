# MAEG4040 Mechatronic Systems

**Course Type**: Major Required
**Units**: 3
**Stream**: Mechatronics (Core)

## Official Description
Integrated design of mechatronic systems. Covers sensors, actuators,
microcontrollers, signal conditioning, and system integration for
mechanical-electrical-computer systems.

## Key Topics
- Sensors: position, velocity, force, temperature
- Actuators: DC motors, stepper motors, servos, pneumatic
- Microcontroller architecture (Arduino, STM32)
- Signal conditioning and filtering
- PWM and motor driver circuits
- H-bridge and motor control
- Data acquisition (DAQ)
- System integration and troubleshooting

## Learning Outcomes
- Design mechatronic system architecture
- Select appropriate sensors/actuators
- Implement microcontroller-based control
- Debug integrated systems
