# MAEG4040 Mechatronic Systems

**Course Type**: Major Elective (Mechatronics Stream)  
**Units**: 3

## Official Description
Mechatronics is at the intersection of mechanical, electronic, control, and software engineering. This course will cover the technologies involved in developing and designing intelligent electro-mechanical systems. Specific topics include physical system modelling and analysis, measurement and manipulation principles, sensors, actuators, data acquisition and conversion, microcontrollers and interface, control system design and tuning. The course includes a significant lab component with a course project in mechatronics system design.

## Key Topics
- Mechatronics system architecture
- Sensors and actuators
- Data acquisition and microcontrollers
- Physical system modelling
- Control system design and tuning
- Integration of mechanical, electronic, and software components
- Hands-on mechatronics project

## Learning Outcomes
- Model and analyze mechatronic systems
- Select and interface sensors and actuators
- Design and tune control systems for mechatronic applications
- Integrate hardware and software in intelligent electro-mechanical systems
