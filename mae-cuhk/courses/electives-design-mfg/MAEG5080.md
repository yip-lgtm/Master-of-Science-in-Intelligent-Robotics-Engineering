# MAEG5080 Smart Materials and Structures

**Course Type**: Major Elective
**Units**: 3
**Stream**: Materials / Design and Manufacturing / Mechatronics

## Official Description
Smart (intelligent) materials that respond to external stimuli, and their
application in adaptive structures. Covers piezoelectric materials, shape
memory alloys, electroactive polymers, and magnetorheological fluids.

## Key Topics
- Piezoelectricity (direct and converse effect)
- Shape memory alloys (SMA) - NiTi (Nitinol)
- Shape memory polymers (SMP)
- Electroactive polymers (EAP, dielectric elastomer)
- Magnetostrictive materials (Terfenol-D)
- Magnetorheological (MR) fluids
- Fiber Bragg gratings (sensors)
- Self-healing materials
- Smart structure applications
- Energy harvesting

## Learning Outcomes
- Understand smart material physics
- Select appropriate smart material for application
- Design smart structures
- Implement sensing and actuation with smart materials
