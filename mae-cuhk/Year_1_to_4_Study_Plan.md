# Year 1–4 Typical Study Plan - BEng Mechanical and Automation Engineering (CUHK)

**Note**: This is a suggested typical plan based on standard CUHK MAE curriculum. Always check the latest official handbook/PDF for your entry year.

## Year 1 (Foundation + Core)
**Semester 1**
- ENGG1110 Problem Solving By Programming (Faculty)
- ENGG1120 Linear Algebra for Engineers (Faculty)
- MATH1510 Calculus for Engineers (Foundation)
- PHYS1110 Engineering Physics (Foundation)
- University Core (English/Chinese/GE/PE)

**Semester 2**
- ENGG1130 Multivariable Calculus for Engineers (Faculty)
- MAEG1020 Computational Design and Fabrication (Foundation)
- MAEG2020 Engineering Mechanics (Major Required)
- Other Foundation / Core courses

**Focus**: Build strong math, physics, and basic programming foundation. Start exploring simple mechanisms.

## Year 2 (Core Mechanics + Circuits + Thermo)
**Semester 1**
- MAEG3010 Mechanics of Materials (Major Required)
- ELEG2202 Fundamentals of Electric Circuits (Major Required)
- MAEG2030 Thermodynamics (Major Required)
- MAEG2601 Technology, Society and Engineering Practice
- University Core + Electives

**Semester 2**
- MAEG3020 Manufacturing Technology (Major Required)
- MAEG3030 Fluid Mechanics (Major Required)
- MAEG3040 Mechanical Design (Major Required)
- Other required / electives

**Focus**: Core mechanical + electrical foundation. Begin linking to mechatronics and simple robot mechanisms. Start simple CAD and prototyping.

## Year 3 (Control + Design + Manufacturing + Early Electives)
**Semester 1**
- MAEG3050 Introduction to Control Systems (Major Required)
- MAEG4030 Heat Transfer (Major Required)
- MAEG3060 Introduction to Robotics (Elective - highly recommended)
- MAEG2050 Robot Development in Practice (Elective)
- Other electives / core

**Semester 2**
- MAEG4040 Mechatronic Systems (Elective)
- MAEG4010 Computer-Integrated Manufacturing (Elective)
- MAEG4020 Finite Element Modelling and Analysis (Elective)
- More depth electives

**Focus**: Control systems, robotics fundamentals, mechatronics integration. Heavy hands-on with simulators and small projects. Build your 3R arm kinematics + PID + state machine here.

## Year 4 (FYP + Advanced Electives + Specialization)
**Semester 1**
- MAEG4998 Final Year Project I (Research Component)
- Advanced electives (e.g. MAEG5080 Smart Materials, MAEG5090 Topics in Robotics, MAEG5160 Design for Additive Manufacturing)
- Remaining core / electives

**Semester 2**
- MAEG4999 Final Year Project II (Research Component)
- More specialized electives
- Final presentations and documentation

**Focus**: Complete FYP (can be extension of your warehouse robot / advanced arm with force control / soft robotics elements). Specialize in Robotics & Automation or Mechatronics stream.

## Recommended Electives for Robotics / Mechatronics Path
- MAEG3060 Introduction to Robotics
- MAEG2050 Robot Development in Practice
- MAEG4040 Mechatronic Systems
- MAEG4050 Modern Control Systems
- MAEG5080 Smart Materials and Structures
- MAEG5090 Topics in Robotics
- MAEG5160 Design for Additive Manufacturing
- ENGG5402 Advanced Robotics (if available)

## Link to Your Personal Projects
- **Year 2–3**: Build foundation → Implement 3R arm kinematics, PID, workspace, force feedback in Pygame
- **Year 3–4**: Integrate state machine, Agent Loop, full warehouse demo
- **FYP**: Physical prototype or advanced simulation with learning / multi-agent systems
