# MW 1.50 – Erection or Alteration of Supporting Structure for Building Services
**狀態**：✅ 收到 3 reference reports

## Reference Cases

### 1. CS222 Waffle Slab (Air-Cooled Magnetic Levitation Unit)
- **Project**: Structural check of waffle slab (2WS1 & 2WS2) 
- **Load**: Air-cooled magnetic levitation unit (3845 kg total)
- **Location**: Strafford House Redevelopment, T.P.I.L. 13, 2/F
- **Complies with**: MW 1.50 Class I Type A
- **Key Details**:
  - Waffle slab: 750×150 mm thick (C45) + rib beams 200×575 mm, span 9.3 m
  - Concrete f_cu = 45 MPa, fy = 460 MPa
  - Plant total = 3845 kg → 3.29 kPa
  - Concrete plinth SDL = 4.6 kPa
  - All checks passed per Hong Kong Concrete Code 2013

### 2. W21027 Cooling Tower (9 Cooling Towers)
- **Project**: Erection of supporting steel frame for 9 nos. air-cooled cooling towers (1500 kg each)
- **Location**: 18 Kwai Hei Street, Kwai Chung (Flat Roof)
- **Complies with**: MW 1.50 Class I Type A (MWTGe.pdf Appendix VII 3.19)
- **Key Details**:
  - Hot-dip galvanized steel frame (UC 152×152×23 kg/m columns + SHS beams)
  - Height < 1.5 m
  - Grade S275, p_y = 275 N/mm²
  - 15 kN/tower, LL per leg = 3.75 kN
  - Wind = 5.52 kPa (Z-dir governing)
  - Hilti HST3-R M12: 92% util
  - All members adequate

### 3. Kai Oi School Steel Frame (Wooden Board)
- **Project**: Supporting steel frame for 90 mm wooden board on assembly hall roof
- **Span**: 9.3 m
- **Key Details**:
  - EA 50×50×5 mm equal angles, simply supported, hot-dip galvanized
  - Grade S275, p_y = 275 N/mm²
  - DL = 0.618 kPa, Notional horizontal = 0.5 kPa
  - Hilti HST3-M10 anchors
